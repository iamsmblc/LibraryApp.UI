## Overview

It is a book borrowing application. It has three functions and two pages for listing, updating and adding.

## Installation

1. Clone the repository: `git clone https://github.com/iamsmblc/LibraryApp.UI.git`
2. Navigate to the project directory: `LibraryApp.UI`
3. Install dependencies: `npm install`
4. Configure settings: Rename `.env.example` to `.env` and update the configuration settings.
